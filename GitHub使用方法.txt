git clone :	把远程仓库clone下来；

cd 文件夹名字操作仓库；

git add . 添加

git commit  -m  "提交信息"  （注：“提交信息”里面换成你需要，如“first commit”）

git log 查看

git push -u origin master   （注：此操作目的是把本地仓库push到github上面，此步骤需要你输入帐号和密码）


工作区  = >  版本区域 
git commit -a  -m ""(上传已经存在的，只是修改)


git reset --hard 3628164   版本回退  id号


git add file //file是我们想要添加的文件


git push命令  https://www.yiibai.com/git/git_push.html


q按键回退 ( git log  )

工作区（本地）查看缓存区

git diff

缓存区 

git diff --cached
 
工作区查版本区域

git diff master
